# money-transfer-api

### To compile and run the app
+ mvn clean compile
+ mvn package
+ java -jar target/jetty-embedded-money-transfer-api.jar


### Endpoints
+ (GET)   /ping
+ (GET)   /acounts/:id
+ (POST)   /acounts
+ (DELETE)   /acounts/:id
+ (GET)   /transactions/:id
+ (POST)   /transactions
+ (DELETE)   /transactions/:id
